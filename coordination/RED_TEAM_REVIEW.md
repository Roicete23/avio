# Red Team Review

## Idea finalista revisada

Landing para captar bares, cafeterias y pequenos restaurantes de Ponteareas + O Porrino que necesiten reposicion simple de consumibles profesionales.

## Objecion 1: "Esto es demasiado parecido a un mayorista"

Riesgo real. Si la web parece un catalogo generico, el cliente comparara solo precio.

Respuesta de marketing: comunicar cercania, rapidez de respuesta, pedido simple y pack a medida. La web debe vender comodidad local, no cantidad de referencias.

## Objecion 2: "El cliente ya tiene proveedor"

Probable. Muchos bares ya compran en mayoristas, supermercados, distribuidores o plataformas.

Respuesta de marketing: no atacar al proveedor actual. Entrar por reposicion puntual, urgencia, pack mensual o segunda opcion fiable.

Mensaje util: "Cuando te falte algo o quieras simplificar una reposicion, escribenos."

## Objecion 3: "Los packs pueden no encajar"

Probable al inicio. Cada bar consume distinto.

Respuesta de marketing: presentar packs como punto de partida, no como obligacion. El Pack A Medida debe estar al mismo nivel visual que los otros.

## Objecion 4: "La web puede parecer demasiado nueva"

Riesgo de confianza si no hay marca conocida.

Respuesta de marketing: usar tono local, contacto directo, zona clara, formulario corto y WhatsApp visible. Evitar parecer una tienda anonima.

## Objecion 5: "Demasiado foco limita oportunidades"

Puede parecer que dejar Vigo o otros sectores fuera reduce mercado.

Respuesta: foco no significa renunciar. La primera web necesita una promesa nitida. Luego se puede duplicar estructura para estetica, alojamientos o Vigo.

## Decision tras red team

Seguir con hosteleria local como primera web, pero con estas protecciones:

- No sonar como catalogo mayorista.
- No prometer mas de lo confirmado.
- Dar mucho protagonismo al Pack A Medida.
- CTA unico dominante: WhatsApp.
- Mensaje de segunda opcion fiable, no de sustitucion agresiva del proveedor actual.
