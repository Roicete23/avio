# Decisions

## Tomadas

- Se trabajara con una carpeta de coordinacion para que el proyecto pueda continuar entre modelos sin perder contexto.
- La primera fase no sera construir una tienda completa, sino validar una propuesta pequena.
- La web debe ser profesional y funcional, pero tambien una herramienta de aprendizaje comercial.
- Se preparara una ronda ciega para Claude antes de cerrar la orientacion final.
- La primera direccion web sera una landing para hosteleria pequena local, empezando por Ponteareas + O Porrino.
- El CTA principal sera WhatsApp y el secundario solicitud de presupuesto.
- No se construira carrito ni pago online en la primera version.
- Codex y Claude se centraran en web y marketing; proveedores, logistica y margenes quedan fuera del foco de estos modelos.

- Nombre de marca confirmado: Avio / Avío. El usuario confirma mantenerlo.
- Tono de marca: local, cercano y profesional. Sin sonar mayorista generico.
- WhatsApp de contacto: 697279292.
- Se mostraran precios orientativos en la web con opcion a Pack A Medida.
- Packs definidos: Pack Basico Bar (~25 EUR), Pack Reposicion Mensual (~55 EUR), Pack A Medida (desde 20 EUR).
- Los precios actuales se mantienen por ahora como orientativos y se revisaran tras hablar con proveedores.
- Se añade un catalogo con carrito de solicitud: el cliente puede seleccionar productos, cantidades y convertirlo en pack personalizado mensual.
- El carrito no cobra ni confirma pedido automaticamente; genera una solicitud por WhatsApp para mantener validacion manual.
