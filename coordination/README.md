# Coordinacion del proyecto

Este directorio guarda el contexto vivo del proyecto de reparto B2B a domicilio para la zona Ponteareas, Vigo, Porrino y cercanias.

Objetivo inmediato: validar si merece la pena construir una propuesta online para vender y tramitar pedidos de consumibles profesionales, sin asumir que la primera idea es la correcta.

Documentos principales:

- `PROJECT_BRIEF.md`: resumen ejecutivo del proyecto.
- `TASK_BOARD.md`: tareas concretas y estado.
- `DECISIONS.md`: decisiones tomadas y pendientes.
- `PROJECT_TRUTHS.md`: hechos que tratamos como verdaderos por ahora.
- `DISAGREEMENTS.md`: desacuerdos entre modelos o criterios.
- `IDEA_GRAVEYARD.md`: ideas descartadas y motivo.
- `SCOPE_GUARDRAILS.md`: limites para no dispersarnos.
- `HANDOFF.md`: contexto para que otro modelo continue.

Regla de trabajo: primero validar una propuesta pequena y vendible; despues construir.
