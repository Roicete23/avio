# Questions For User

## Prioridad alta

1. Teneis ya proveedor mayorista o solo la idea?
2. Vais a tener stock propio, comprar bajo pedido o combinar ambas cosas?
3. Que vehiculo y disponibilidad real teneis para repartir?
4. Cual seria vuestra zona inicial mas realista durante las primeras 2-4 semanas?
5. Que sector os motiva mas vender: hosteleria, estetica/spa, alojamiento, clinicas u otro?
6. Teneis algun contacto real en negocios locales para preguntar esta semana?
7. Cual seria el objetivo minimo para considerar que la prueba funciona?

## Prioridad media

1. Queremos una marca con nombre ya elegido o una marca provisional?
2. Preferis pedidos por WhatsApp, formulario web, llamada o carrito?
3. Habeis calculado margen minimo por pedido?
4. Que productos puede servir el proveedor con mejores precios o condiciones?
5. Que plazo de entrega podeis prometer sin tension?

## Para la web

1. Debe sonar mas premium, local, barato, rapido o profesional?
2. Quereis mostrar precios desde el inicio?
3. Quereis que el primer CTA sea "Pedir presupuesto", "Hacer pedido", "Hablar por WhatsApp" o "Ver packs"?
